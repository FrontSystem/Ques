神奇的DIY组件[unstable]

DIY组件是通过markup的方式，引入Nodejs插件，使得Nodejs可以对该页面进行处理。

制作一个diy-preload组件

新建/src/components/diy/preload/main.js

module.exports = function (opts) { // 拿到组件的容器，然后replaceWith成下面脚本 opts.container.replaceWith('<script>alert("Just do it yourself!");</script>'); };

引入diy-preload组件

<diy-preload></diy-preload>

则页面将会像该页面一样打印Just do it yourself!

可能的应用场景

  • CGI预加载方案，例如markup一个<diy-preload>则自动扫描脚本分析出需要预加载的CGI，然后打入页面中
  • 通用页面处理，打点上报等等
